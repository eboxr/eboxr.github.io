# eBoxr.com

Portfolio website for **eBoxr** — a small studio crafting thoughtful iOS and web applications.

## Tech Stack

- HTML5 with Bootstrap 5.3.8 (dark theme)
- Bootstrap Icons 1.13.1
- Google Fonts (Inter)
- Custom CSS + vanilla JS
- No build tools, no npm — just open `index.html`

## Structure

```
index.html              Main SPA (hash-routed virtual pages)
css/style.css           Design tokens + custom styles
js/main.js              Router, navigation, animations
assets/images/          Avatars, screenshots, icons
card-wallet/            Rewards Card Wallet support pages
  privacy-policy.html
  support.html
```

## Development

Open `index.html` in a browser. No build step required.
